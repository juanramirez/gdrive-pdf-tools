# gdrive-pdf-tools

A small collection of Bash tools to bulk-process PDFs stored in Google Drive.

## Included tools

- **Deskew + OCR**
  See: [docs/autodeskew.md](docs/autodeskew.md)

- **Filename normalization (dates)**
  See: [docs/normalize-names.md](docs/normalize-names.md)

## Philosophy

Small, composable, idempotent Unix-style tools.
